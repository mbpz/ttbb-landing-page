    ### 技术趋势
    
    1. **MCP 协议快速标准化**  
       微软、Oracle 等大厂推出官方 MCP 服务器实现，"工具即服务"模式蔚然成风。PowerPoint、dbt、FreeCAD、Teams 等专业领域工具正加速接入 MCP 生态。
    
    2. **Agent 框架走向垂直细分**  
       通用框架（如 AutoAgent）追求零代码体验的同时，垂直领域框架涌现：OrcaLoca 专注软件缺陷定位、TrajAgent 聚焦轨迹建模、cadCAD-GPT 面向 token 系统仿真。

       Python 仍占主流（12/15 个 LLM Agent 项目），但 C++ (humanus.cpp)、TypeScript (Overture, Ace-Mcp-Node)、Go (mcptools)、C# (unity-mcp) 等多语言并存的格局日趋明显。
    
    4. **安全与可视化成为新焦点**  
       awesome-mcp-security 关注 MCP 安全风险；Overture 提供 Agent 执行计划的可视化流程图，填补"黑盒调试"痛点。
